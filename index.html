Material-Web login form

A test login page to check if Password autofill works well

This page is built by following these steps : material-web.dev/about/quick-start.

The source code is available (see index.html, index.js and the Github-CI workflow file that builds and publishes it here)

Material-web makes usage of Shadow-DOM around the fields.

This could be the root cause of some password autofillers, can't manage to detect them.
